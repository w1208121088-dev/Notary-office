# 投机倒把交易所 — 启动说明

## 第一步：安装依赖

打开终端（VS Code 里按 Ctrl+` 可以打开），进入项目文件夹，输入：

```
pip install fastapi uvicorn websockets
```

## 第二步：启动服务器

```
uvicorn server:app --host 0.0.0.0 --port 8000 --reload
```

看到 "Application startup complete." 就说明成功了。

## 第三步：打开游戏

- 你自己：浏览器打开 http://localhost:8000
- 同一局域网的朋友：打开 http://你的IP:8000
  （查看你的IP：Windows 终端输入 ipconfig，找 IPv4 地址）

## 规则说明

- 每位玩家初始拥有 100,000 钞
- 稀有金属按 kg 交易，可输入小数
- 缝纫机兔按整只交易
- 买入拉高价格，卖出压低价格
- 每 10 秒市场会自动小幅随机波动
- 价格最低跌至基底价的 20%，最高涨至基底价的 500%
